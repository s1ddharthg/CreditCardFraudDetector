## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd credit-card-fraud-detection
   ```

2. Install the required dependencies:
   ```bash
   pip install pandas==2.2.3 scikit-learn==1.6.0 matplotlib==3.9.3 kagglehub[pandas-datasets]
   ```

3. Open the Jupyter notebook:
   ```bash
   jupyter notebook "Credit Card Fraud Detection Model.ipynb"
   ```

## Usage

1. Launch Jupyter Notebook and open `Credit Card Fraud Detection Model.ipynb`.
2. Execute the cells to load data, preprocess features, train models, and evaluate fraud detection performance.
3. Adjust parameters or datasets for custom analysis.

## Features

- Data preprocessing with normalization and scaling
- Model training using Decision Tree and Support Vector Machine classifiers
- Handling imbalanced datasets with sample weights
- Performance evaluation using ROC AUC score
- Visualization of results with Matplotlib
